ESS CS-Studio Product Changelog

Refer to the Compatibility Notes and Known Bugs page for more information.

Version 4.5.7.0 - 2018-03-08

  • Templates for Display Builder: ess-templates
  • Better DB performances after cacheing colours and fonts.
  • DB: Fixed incorrect macro expansion if macro has default value.
  • DB: Fixed Action Button widget text alignment.
  • DB: Fixed named fonts not loaded correctly.
  • DB: Fixed Group Widget height and width not saved correctly.
  • DB: Fixed standalone windows not sized correctly.
  • DB: Fixed Picture widget not smoothing on rescale.
  • BOY: MEDM conversion improved.
  • Based on CS-Studio CE 4.5.7-SNAPSHOT
  • Closed Issues

Version 4.5.6.1 - 2018-02-13

  • Improved workflow and usability in Rules dialog.
  • Added rotation to Symbol and Text Symbol widgets.
  • Changed logging preferences.
  • More badge icons for vacuum.
  • Based on CS-Studio CE 4.5.6-SNAPSHOT
  • Closed Issues

Version 4.5.6.0 - 2018-01-31

  • New Thumb Wheel widget (back-compatible with BOY's one).
  • Performance improvements and bug fixes in Gauge and Knob widgets.
  • Light (default) and Dark ESS colour themes.
  • Added search capabilities in Properties and Palette.
  • New colour and font selectors.
  • Improved workflow and usability in Scripts dialog.
  • Based on CS-Studio CE 4.5.6-SNAPSHOT
  • Closed Issues
